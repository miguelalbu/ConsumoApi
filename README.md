<h1>Projeto Consultando CEP</h1>
<p>Foi ultilizado a API CiaCep, com as tecnologias HTML, CSS, JS.</p>

# Funcionalidades
- Busca endereço de um CEP

# Como usar
- Digite o CEP
- Busque pelo CEP
- Tenha todas as informações

# Observações
- O CEP deve ter 8 dígitos
- Deve ser um CEP válido
- Projeto utiliza API ViaCep
